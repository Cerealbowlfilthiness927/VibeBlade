## ⚙️ Understanding How It Works

VibeBlade uses technology designed for home computers.

*   **CPU Inference:** The software uses your main computer processor. This allows you to run models even if you do not own an expensive graphics card.
*   **Quantization:** This shrinks the size of complex models. You get similar quality output while using significantly less memory. 
*   **Memory Management:** The program splits chunks of the model between your processor and your system RAM. This prevents crashes on systems with limited memory.
*   **Expert Offloading:** The software identifies which parts of a model matter most. It prioritizes these to ensure quick responses.

## 📖 Troubleshooting Common Issues

*   **Slow Responses:** Your processor might be busy with other tasks. Close heavy applications like video editors or web browsers with many open tabs to free up resources.
*   **Program Crashes:** You might attempt to run a model that is too large for your RAM. Choose a smaller "quantized" version of the model from the download library. 
*   **Download Failures:** Check your internet connection. Ensure you have enough disk space for the model file.
*   **Graphics Errors:** Update your Windows display drivers. Contact your hardware manufacturer website to find the newest update for your specific system.
